## Controls

###Debug
* F1: start level 1
* F2: start level 2
* F3: restar level
* F5: save game
* F6: load game
* F7: show/hide colliders
* F8: godmode

###Gameplay
* W: climb
* A: left
* S: slide
* D: right
* LEFT SHIFT: run
* SPACE: jump
* "While in godmode use W,A,S,D to move in all directions.
